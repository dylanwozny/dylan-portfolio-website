React To Do App

This is a Asynchronous CRUD Web App that stores each users to do list. Built with React and Cloud Firestore. Functionality of this web app includes: Client and server side validation with easy user experience error messages. Sign in options include a google provider and standard email. Give it a try and sign in with email, no need to provide a real one. You can also login with the google provider.
